# Become OverMan - React Frontend

Простое и надежное React-приложение с JWT авторизацией.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Структура

- `src/api.js` - API клиент с автоматическим добавлением JWT токена
- `src/App.jsx` - Главный компонент приложения
- `src/components/` - Компоненты для различных вкладок:
  - `AuthTab.jsx` - Авторизация и регистрация
  - `ProfileTab.jsx` - Профиль пользователя
  - `QuestsTab.jsx` - Список доступных квестов
  - `MyQuestsTab.jsx` - Мои квесты (активные и завершенные)
  - `FriendsTab.jsx` - Управление друзьями

## Особенности

- JWT токен автоматически сохраняется в localStorage
- Токен автоматически добавляется во все API запросы
- Простая и надежная система авторизации
- Все функции работают так же, как в index.html версии

## API

Бэкенд должен быть запущен на `http://localhost:8080`
